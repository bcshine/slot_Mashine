단일화면으로 이루어진 "💎 행운의 쿠폰 게임 💎" 만든다. 
화면은 3부분으로 구분된다. 

상단부분
타이틀: "💎 행운의 쿠폰 게임 💎"
설명: "주문하신 동안 행운을 돌려보세요!"

중단부분 - 실제 카지노 슬롯머신 디자인 ✨

=== 완성된 디자인 특징 ===
- "💎 JACKPOT 💎" 헤더 (금색 글씨)
- slot.png 이미지를 배경으로 한다. 
- slot.png 이미지안의 3개 흰색 릴, 이 릴에서,  
- 실제 슬롯머신처럼 숫자가 위에서 아래로 흐르는 애니메이션
- 3D 그림자 효과와 입체감
- 숫자 폰트는 두껍고 잘 어울리는 폰트 사용할 것

하단 부분
시작 버튼 (3D 효과, 호버 애니메이션)

기능 및 애니메이션
- 시작 버튼 클릭 시 릴이 5초간 회전 (3초 고속 + 2초 감속)
- 실제 슬롯머신처럼 위에서 아래로 흐르는 애니메이션
- 무제한 플레이 가능 (하루 1회 제한 해제됨)
- 멈추면서 선물 결과 표시

선물구성 (확률 시스템):
릴 3개 (회전 바퀴)
결과:
50-50-50 : 50% 할인 (확률 2%) - 최고 잭팟!
30-30-30 : 30% 할인 (확률 10%)
20-20-20 : 20% 할인 (확률 18%)
10-10-10 : 10% 할인 (확률 20%)
Cup-Cup-Cup : 음료수 (확률 10%)
Sal-Sal-Sal : 샐러드 (확률 10%)
Soup-Soup-Soup: 스프 (확률 10%)
천-천-천 : 천원할인 (확률 10%)
일치하지 않을 경우 : 꽝, 아깝습니다! (확률 10%)

디자인 스타일:
- 실제 카지노 슬롯머신 외관
- 단정하고, 편안한색, 고급스러운 색감
- 다이아몬드 장식과 화려한 효과
- 깜빡이는 LED 불빛 효과

선물이 나올 경우:
- 선물내용을 화려한 쿠폰 형태로 표시
- "🎉 축하합니다! 🎉" 메시지
- 쿠폰 애니메이션 효과 (반짝임, 펄스)

기술 사양:
- 반응형 디자인 (모바일/PC 대응, 최대 640px)
- 넥슨고딕체 폰트 전체 적용
- localStorage를 통한 쿠폰 관리
- 단일 HTML 파일로 구성 (의존성 없음)

결과화면 기능:
버튼: [쿠폰 저장하기] [홈으로]
- 쿠폰은 티켓 형태로 예쁘게 표시
- 쿠폰 유효기간 표시 (발급일로부터 30일)
- 쿠폰 저장 및 관리 기능

✅ 넥슨고딕체 폰트 전체 적용
✅ 실제 슬롯머신 애니메이션 (위→아래 흐름)
✅ 깜빡이는 불빛과 다이아몬드 장식
✅ 3D 효과와 그림자로 입체감 구현
✅ React 기반의 현대적 구조
✅ 완전한 반응형 디자인

파일 구성:
- index.html (메인 파일, React CDN 사용)
- 모든 CSS와 JavaScript 내장
- 외부 의존성 없음
- 바로 실행 가능


수정요청사항
첫화면 맨  아래 비어있는 부분에, 다음의  메시지를 적는다. 
* 숫자는 할인율 % 입니다
* 1등 50% 할인 ~ 10% 할인까지
* 음료수, 샐러드, 스프, 천원할인까지
* 물론 꽝도 있습니다
* 행운에 도전해보세요!

- 선물당첨될경우 "축하합니다!" 메시지가 크게 나온다
- 이경우 3초간 ddd.wav 나온다
- 꽝일 경우, ggg.png 좌에서 우로 4초간 날라가고, ggg.wav 소리를 내보낸다. 

결과표시될때, 
쿠폰받기, 다시도전  버튼이 있어야 한다. 
이후, 쿠폰받기를 누르면, 쿠폰이 화면에 보여야 한다. 
그 아래에, 쿠폰 저장하기 버튼, 다시도전 버튼이 나와야 한다. 
이후, 쿠폰저장하기 버튼을 누르면, 쿠폰이실제 저장되어야  한다.
